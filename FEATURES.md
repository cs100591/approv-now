# Approv Now - 功能介绍

## 🎯 应用定位

**Approv Now** 是一款移动优先的多工作区审批引擎，帮助企业快速构建和管理工作流审批系统。支持从简单的请假申请到复杂的多级采购审批。

---

## 🚀 核心功能

### **1. 🤖 AI智能模板生成**

**特色**: 业界领先的AI驱动模板创建

**功能**:
- 输入自然语言描述，AI自动生成完整模板
- 15种预设场景覆盖常见业务流程
- 智能字段推荐和审批流程设计
- 本地优先匹配 + DeepSeek AI生成
- 成本比OpenAI低50%

**示例**:
```
用户输入: "员工请假流程，需要经理和HR审批"
AI生成:
- 字段: 请假类型、开始日期、结束日期、请假天数、请假原因
- 审批: Level 1 (经理) → Level 2 (HR，超过3天时)
```

**支持的模板类型**:
1. **人力资源**: 请假、加班、入职、离职、招聘
2. **财务审批**: 费用报销、采购申请、付款申请、预算审批
3. **行政管理**: 出差申请、用车申请、资产领用
4. **项目管理**: 项目立项、合同审批
5. **通用场景**: 自定义审批流程

## 💼 使用场景

### **场景1: 中小企业OA**
**痛点**: 传统纸质审批效率低，无法追踪进度  
**解决方案**: Approv Now提供数字化审批流，手机随时审批，进度实时可见

### **场景2: 连锁门店管理**
**痛点**: 多门店采购、报销需要总部审批，流程慢  
**解决方案**: 多工作区管理，各门店独立数据，总部统一审批

### **场景3: 项目制公司**
**痛点**: 项目预算、采购、报销需要多级审批  
**解决方案**: 项目级工作区，预算控制，自动路由审批

### **场景4: 远程团队协作**
**痛点**: 分布式团队审批难，邮件沟通混乱  
**解决方案**: 移动端优先，推送通知，PDF存档

## 🛠️ 技术特色

### **架构设计**
```
模块化架构 (14个独立模块)
├── Auth (认证)
├── Workspace (工作区)
├── Template (模板)
├── Request (请求)
├── Approval Engine (审批引擎)
├── Revision (版本控制)
├── Export (PDF导出)
├── Verification (验证)
├── Subscription (订阅)
├── Notification (通知)
├── Analytics (分析)
├── Plan Enforcement (计划限制)
├── Workspace Switch (工作区切换)
└── Permission Guard (权限控制)
```

### **技术栈**
- **前端**: Flutter 3.x (跨平台)
- **后端**: Firebase (无服务器架构)
- **数据库**: Cloud Firestore (实时同步)
- **认证**: Firebase Auth
- **存储**: Firebase Storage
- **AI**: DeepSeek API
- **PDF**: 原生生成

### **优势**
1. **快速部署**: 无需后端开发，Firebase即开即用
2. **跨平台**: 一套代码，iOS/Android/Web全覆盖
3. **实时同步**: Firestore实时数据同步
4. **AI驱动**: 智能模板生成，降低配置成本
5. **可扩展**: 模块化设计，易于扩展功能

## 🚀 快速开始

### **5分钟部署**
```bash
# 1. 克隆项目
git clone https://github.com/your-org/approve-now.git

# 2. 安装依赖
cd approve-now
flutter pub get

# 3. 配置Firebase
# - 创建Firebase项目
# - 下载配置文件
# - 放置到 ios/Runner 和 android/app

# 4. 运行
flutter run
```

### **首次使用**
1. 注册账号 → 自动创建默认工作区
2. 点击 "Templates" → "Create Template"
3. 输入 "请假申请" → AI自动生成模板
4. 点击 "New Request" → 提交审批
5. 查看 "My Approvals" → 审批请求
